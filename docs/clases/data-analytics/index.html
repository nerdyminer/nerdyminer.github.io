Una ruta elemental

Esta ruta reúne clases orientadas a construir una base práctica y sólida para trabajar con datos en Python. El foco está puesto en aprender a manipular estructuras tabulares, vectorizar cálculos, construir visualizaciones eficaces, desarrollar pequeños productos analíticos y utilizar herramientas científicas que permitan pasar desde el procesamiento de datos hacia el modelamiento. La idea no es sólo usar librerías, sino entender qué problema resuelve cada una, en qué contexto conviene emplearla y cómo integrarlas en flujos de trabajo reproducibles.
