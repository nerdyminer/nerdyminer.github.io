Introducción al análisis de datos en Python

← Volver a Data & Analytics

Python, estructuras tabulares, cómputo vectorizado y procesamiento escalable

Este bloque reúne las herramientas fundamentales para trabajar con datos en Python desde una perspectiva moderna y aplicada. El recorrido parte con el cómputo vectorizado y el trabajo con arreglos por medio de Numpy, continúa con la manipulación tabular y el análisis exploratorio usando Pandas, y luego se expande hacia enfoques más recientes de procesamiento columnar con Polars y flujos distribuidos con PySpark. La idea no es sólo aprender librerías, sino entender cómo se organiza un flujo analítico completo: cargar, transformar, explorar, modelar y escalar datos de manera reproducible.

De arreglos a pipelines analíticos

Cada uno de los siguientes módulos desarrolla una capa distinta del trabajo con datos en Python: Desde la base numérica del cómputo vectorizado hasta el procesamiento tabular moderno y los entornos distribuidos.

Módulo 1

Computación vectorizada y arreglos con Numpy

Bloque dedicado a introducir la lógica de cómputo científico en Python por medio de arreglos multidimensionales y operaciones vectorizadas de alta eficiencia.

  • Arreglos, geometría y tipos de datos
  • Creación, indexación y slicing
  • Funciones universales y broadcasting

Módulo 2

Manipulación tabular y análisis con Pandas

Bloque centrado en estructuras tabulares, limpieza de datos, transformación de columnas y construcción de análisis exploratorios reproducibles.

Módulo 3

Procesamiento columnar y analítica moderna con Polars

Bloque orientado a explorar una alternativa moderna de alto rendimiento para procesamiento tabular, con énfasis en expresiones, lazy execution y eficiencia.

Módulo 4

Datos distribuidos y escalamiento con PySpark

Bloque introductorio al trabajo con datos a escala, usando el ecosistema de Spark desde Python para procesamiento distribuido y construcción de pipelines analíticos.

  • Qué es Spark y cuándo usarlo
  • DataFrames distribuidos y transformaciones

  • Casos de uso en pipelines de datos
